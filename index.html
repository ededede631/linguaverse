支持的功能

  • 英语 / 日语 / 韩语 三种语言
  • 初级 / 中级 / 高级 三个阶段
  • 单词记忆、语法练习、口语跟读、听力训练、拼写手写
  • 本地学习记录、生词本、错题复盘、每日打卡
  • 离线缓存（需先访问过一次）

如何启用

  1. 在浏览器设置中启用 JavaScript
  2. 刷新页面即可正常使用
